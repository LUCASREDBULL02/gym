# Bebi Gym v13.3 – Full App

Den här versionen innehåller:

- Enkel avatar (endast en bild) för Maria Kristina.
- Dashboard med:
  - XP & Battle Pass tier.
  - Muskelkarta kopplad till strength standards.
  - Boss Raid-kort med faser.
- Loggning via modal (vikt + reps).
- PR-system per övning med lista.
- Program Runner med färdiga program.
- Achievements (flera nivåer).
- Battle Pass med rewards & claim-knappar.
- Delete logg + full omräkning av:
  - XP
  - Battle tier
  - Boss-HP
  - PR-map
  - Muskelkarta.

## Kom igång

```bash
npm install
npm run dev
```

Avatar-bilden finns i `src/assets/avatar.png`. Byt gärna ut den mot din egen PNG (behåll filnamnet).
